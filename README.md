# tofn
Typed ordered fuzzy numbers in OCaml.  Based on the work in [Rings of Typed Ordered Fuzzy Numbers](https://arxiv.org/abs/2010.07764).

Documentation is hosted [here](http://mkukla.net/doc/tofn/Tofn.html).
